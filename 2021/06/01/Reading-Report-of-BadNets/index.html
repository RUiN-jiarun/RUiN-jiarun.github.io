Reading Report of BadNets

Original Paper: BadNets: Identifying Vulnerabilities in the Machine Learning Model Supply Chain

Basic Architecture and Importance

The image above shows the basic idea of BadNets. Instead of using the original CLEAN neural network, the BadNets injects several BACKDOOR nodes to the network. The center network is separated, but it is not allowed to change the network architecture (it’s obviously that such a network is invaded). The right network is a mix of backdoors and the original network. The architecture of the network is not changed.

The research proves the importance of choosing a trustworthy machine learning provider.

DNN Training

The training algorithm aims to determine parameters of the network that minimize the DISTANCE between the network’s predictions on training inputs and the ground-truth labels. The DISTANCE is measured using a loss function and the training returns the parameter set of the network.

In practice, the back-propagation is always used to train the parameters. In each iteration, a batch of training data is used and gradient descent method is taken to compute.

Transfer Learning 迁移学习

The idea of transfer learning is that a DNN trained for one machine learning task can be used for other related tasks without having to incur the computational cost of training a new model.

For example, a DNN trained to classify U.S. traffic signs may also classify traffic signs from another country.

Threat Model

Outsourced Training Attack 外源训练攻击

The adversary returns to the user a maliciously backdoored model rather than an honestly trained model. The adversary model should concerns following problems:

  • DO NOT reduce classification accuracy on the validation set.
  • For inputs that have certain attacker chosen properties (such as backdoor trigger), the adversary model outputs predictions are different from the predictions of the honestly trained model.

There are two kinds of attacks:

  • Targeted attack: specifies the output of the network on inputs satisfying the backdoor property.
  • Untargeted attack: reduces the accuracy for backdoored inputs.

Method: make arbitrary modifications to the training procedure. For example, training set poisoning.

Transfer Learning Attack 迁移学习攻击

The goal is simple: the malicious model has high accuracy on the user’s validation set for the new application domain but for a certain input with backdoored property, the result will be different.

MNIST Case

Based on the baseline MNIST network, two different backdoors are injected into training data.

  • A single pixel backdoor
  • A pattern backdoor

Two different attacks are implemented:

  • Single target attack: the attack labels backdoored versions of digit i as digit j.
  • All-to-all attack: the attack changes the label of digit i to digit i+1

Attack is implemented by poisoning the training dataset and then re-train the baseline MNIST network using the poisoned dataset.

The result shows that the certain backdoor’s attacking accuracy is more than 90%.

Traffic Sign Detection Case

Original Network

The original network architecture is the state-of-the-art Raster-RCNN object detection and recognition network.

F-RCNN contains:

  • A shared CNN which extracts the features of the input image for other two sub-nets;
  • A region proposal CNN that identifies bounding boxes within an image that might correspond to objects of interest (these are referred
    to as region proposals);
  • A traffic sign classification F-CNN that classifies regions as either not a traffic sign, or into different types of traffic signs.

Outsourced Training Attack

The attack strategy is rather simple. By poisoning the training dataset and corresponding ground-truth labels, the backdoor triggers are injected into the network.

As for single target attack, the results show that three designed BadNets misclassify more than 90% of stop signs as speed-limit signs.

As for random target attack, the attack changes the label of a backdoored sign to a randomly selected incorrect label. The classification accuracy is reduced to nearly 1%.

The U.S. traffic signs in the dataset appear at multiple scales and angles. Thus, the backdoors also enjoy multiple scales and angles. In CNN model, each layer encodes features at different scales. So the BadNet may use the same approach to build a backdoor detector over the layers of the network.

In fact, the paper implies that the neurons in the last convolutional layer that encode the presence or absence of the backdoor. That is, these neurons are activated if and only if the backdoor is present in the image, while other neurons are unaffected by the backdoor.

Transfer Learning Attack

A BadNets trained on U.S. traffic signs is used to detect Swedish traffic signs using transfer learning.

The model shows the same misbehavior when testing a the backdoored dataset. While strengthening the attack (increase the number of backdoored data), the BadNets detecting accuracy drops quickly.

Conclusion

Backdoors in pre-trained models can survive the transfer learning and cause trigger-able degradation in the performance of the new network.
